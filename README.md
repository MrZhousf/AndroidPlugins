# AndroidPlugins
Android Studio 插件，快速生成代码。

## 功能点
* Double Check 单例生成
* 确认对话框生成
* 后续优化中...

## 引用方式
* 1.检出AndroidPlugins项目
* 2.复制项目根目录下的AndroidPlugins.jar到C盘（自定义）
* 3.打开Android Studio -> Settings -> Plugins -> Install plugin from disk
* 4.选择AndroidPlugins.jar
* 5.重启Android Studio

## 项目源码
项目中已包含所有支持业务的源码，可以进行自定义插件编写。
